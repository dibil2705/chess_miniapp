Правила расчета Elo для турниров

1. Каждый новый игрок начинает турнир с Elo 1500.
2. K-фактор равен 32.
3. Elo считается только по сыгранным партиям.
4. Bye не влияет на Elo.
5. Турнирные очки и Бухгольц не смешиваются с Elo.
6. Очки и Бухгольц используются для мест внутри турнира.
7. Elo используется только как отдельный показатель силы игрока.

Порядок расчета

1. Партии считаются строго по турам: 1 -> 2 -> 3 -> 4 -> 5.
2. Внутри одного тура все партии считаются от рейтингов игроков на начало тура.
3. После расчета всех партий тура изменения Elo применяются одновременно.
4. Затем начинается расчет следующего тура.
5. Итоговый Elo округляется до целого числа.

Формула

expectedScore = 1 / (1 + 10 ** ((opponentRating - playerRating) / 400))

newRating = oldRating + 32 * (score - expectedScore)

Значения score

Победа = 1
Ничья = 0.5
Поражение = 0

Пояснение

Если игрок стартует с Elo 1500 и играет против соперника с Elo 1500,
его ожидаемый результат в партии равен 0.5.

При победе игрок получает примерно +16 Elo.
При поражении игрок теряет примерно -16 Elo.
При ничьей против равного соперника изменение примерно 0 Elo.
